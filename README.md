# ArcSplit Pro ⚡

**Professional Token Splitter on Arc Testnet**

A beautiful and fast tool to split tokens between multiple addresses with custom percentages.

## ✨ Features

- Modern Glassmorphism UI + Dark/Light Mode
- Split tokens by percentage
- Quick templates (50-50, 33-33-34, etc.)
- Equal Split with any number of recipients
- Distribute Remaining Percentage
- Supports USDC, EURC, cirBTC
- Transaction history & export
- Fully responsive

---

## 🚀 How to Use

1. Go to [ArcSplit Pro](https://arcsplit.kamkazi-1297.workers.dev/)
2. Connect MetaMask (auto switches to Arc Testnet)
3. Select token and amount
4. Add recipients and percentages
5. Click **Send Split Transaction**
